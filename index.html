Connect Google Sheets
Free, no API key needed — uses Google Apps Script
1
Create your Google Sheet
Go to sheets.google.com and create a new spreadsheet. Name the first sheet tab (e.g. Tools). Paste this header row in Row 1:
Name Uses Features Specialization Website Category Pricing Added
2
Add an Apps Script
In your Sheet: Extensions → Apps Script. Replace all code with this:
const SHEET = "Tools"; // change to your sheet tab name function doPost(e) { try { const d = JSON.parse(e.postData.contents); const sheet = SpreadsheetApp .getActiveSpreadsheet() .getSheetByName(SHEET); sheet.appendRow([ d.name, d.uses, d.features, d.specialization, d.website, d.category, d.pricing, new Date().toLocaleString() ]); return ContentService .createTextOutput(JSON.stringify({ok:true})) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService .createTextOutput(JSON.stringify({ok:false,err:err.toString()})) .setMimeType(ContentService.MimeType.JSON); } }
3
Deploy as Web App
Click Deploy → New deployment
Type: Web app
Execute as: Me  |  Who has access: Anyone
Click Deploy → authorize → copy the Web app URL
4
Paste URL in the sidebar
Back in NeuralVault, paste the Web app URL into Apps Script Web App URL and type your sheet tab name. Every new tool you add will now sync to your spreadsheet automatically. ✓
